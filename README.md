# 💼 Job Tracker PRO

A modern job application tracker built with **HTML, CSS, and JavaScript**.

Job Tracker PRO helps users organize their job search process by tracking applications, interviews, offers, and rejections in one clean dashboard.

---

## ✨ Features

- ➕ Add new job applications
- 🔄 Update application status instantly
- 🔍 Search applications by company or position
- 🏷 Filter applications by status:
  - Applied
  - Interview
  - Offer
  - Rejected
- 📊 Real-time statistics dashboard
- 🗑 Delete applications
- 💾 Persistent data with LocalStorage
- 📱 Fully responsive design
